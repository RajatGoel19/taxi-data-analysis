# Taxi Trip Data Analysis

Exploratory data analysis (EDA) of taxi trip data, worked through as a series of question-driven Jupyter notebooks using Python and pandas. Each notebook tackles one analytical question about the dataset and produces the supporting tables and charts.

## Overview

This project explores a taxi trip dataset to surface patterns in trips, fares, distances, durations, and related fields. The work is split across several numbered notebooks, each answering a specific question and building up a complete picture of the data — including at least one distribution study rendered as a histogram.

## Repository structure

| Path | Description |
| --- | --- |
| **Q 1.ipynb** … **Q 11.ipynb** | Individual notebooks, each answering one analytical question about the dataset |
| **Q 6 histogram.ipynb** | Distribution analysis with a histogram visualization |
| **Images/** | Exported charts and figures generated by the notebooks |

## Tech stack

- **Language:** Python 3
- **Environment:** Jupyter Notebook
- **Core libraries:** pandas, NumPy, Matplotlib

## Getting started

Clone the repository:

    git clone https://github.com/RajatGoel19/taxi-data-analysis.git
    cd taxi-data-analysis

Install the core libraries (a virtual environment is recommended):

    pip install pandas numpy matplotlib jupyter

Launch Jupyter and open any of the **Q\*.ipynb** notebooks:

    jupyter notebook

## Notes

The notebooks are self-contained and intended to be read in numerical order; each focuses on a single question in the analysis.
